# emlog-wxmini

#### 介绍
emlogPro小程序使用uni-app开发，适配安卓、iOS、小程序

#### 安装教程

1.  导入项目到HBuilderX
2.  修改setting.js内的配置信息
3.  注册账号（http://tp.hkiii.cn）
4.  联系我激活账号即可
